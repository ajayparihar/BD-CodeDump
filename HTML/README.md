# Constitution of India - Sources Visualization

This project provides an interactive visualization of the various sources from which the Constitution of India has borrowed its features. It parses a CSV dataset of articles and sources, aggregates the data, and displays it using a bar chart.

## Features

-   **Visual Overview**: A bar chart showing the number of articles borrowed from each nation/source.
-   **Percentage Stats**: Displays the percentage contribution of each source.
-   **Interactive Exploration**: Click on any source in the stats table to expand and view the specific list of articles and their content borrowed from that source.

## File Structure

-   `index.html`: The main web page containing the Chart.js visualization and interactive table.
-   `generate_details_file.py`: A Python script that processes the CSV data and generates the JSON file used by the visualization.
-   `source_details.json`: The structured JSON data file containing source-to-article mappings (generated by the Python script).
-   `Untitled spreadsheet - Sheet3.csv`: The input CSV file containing raw data with columns for Article, Content, and Source.

## Setup and Usage

1.  **Prerequisites**:
    -   Python 3.x installed on your system.
    -   A modern web browser.

2.  **Data Generation**:
    If you have updated the CSV file (`Untitled spreadsheet - Sheet3.csv`), you need to regenerate the JSON data:
    ```bash
    python generate_details_file.py
    ```
    This will create/update `source_details.json`.

3.  **Viewing the Visualization**:
    -   Simply open `index.html` in your web browser.
    -   No web server is required; it works directly from the file system (though some browsers might restrict local file fetch requests, in which case using a simple local server like `python -m http.server` is recommended).

## Data Format

The input CSV is expected to have the following columns (in order):
1.  **Article**: The article number.
2.  **Content / Feature**: The description of the article.
3.  **Source / Borrowed From**: The country or source name (can contain multiple sources separated by commas).

## Technologies Used

-   **HTML/CSS/JavaScript**: Frontend visualization.
-   **Chart.js**: Charting library for the bar graph.
-   **Python**: Data processing and transformation.
