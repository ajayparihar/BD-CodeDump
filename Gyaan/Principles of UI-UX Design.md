## Principles of UI/UX Design

### Usability (6)
1. **Visibility** - System status always visible
2. **Match Real World** - Familiar language and concepts
3. **User Control** - Freedom to undo/redo
4. **Consistency** - Same elements behave same way
5. **Error Prevention** - Design that prevents problems
6. **Recognition over Recall** - Show options, don't require memory

### Design Fundamentals (6)
7. **Hierarchy** - Visual importance guides attention
8. **Proximity** - Related items grouped together
9. **Contrast** - Difference creates emphasis
10. **Alignment** - Elements visually connected
11. **Repetition** - Consistent patterns build unity
12. **White Space** - Breathing room improves readability

### Interaction (6)
13. **Affordance** - Elements suggest their function
14. **Feedback** - Actions produce visible results
15. **Mapping** - Controls relate logically to effects
16. **Constraints** - Limit options to prevent errors
17. **Forgiveness** - Easy recovery from mistakes
18. **Discoverability** - Features can be found naturally

### Accessibility (6)
19. **Perceivable** - Information presented so all can perceive
20. **Operable** - Interface works with various inputs
21. **Understandable** - Content readable by all users
22. **Robust** - Works with assistive technologies
23. **WCAG Color Contrast** - Sufficient contrast ratios
24. **Keyboard Navigation** - Full functionality without mouse
